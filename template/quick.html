    Bubble Sort

    Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. It passes through the list multiple times until no more swaps are needed, indicating that the list is sorted.

    Description

    1. Comparison and Swap: Bubble sort compares adjacent pairs of elements in the list. If they are in the wrong order (e.g., smaller followed by larger), it swaps them.
    2. Passes through the list: It makes multiple passes through the list. After each pass, the largest unsorted element 'bubbles up' to its correct position.
    3. Repeat until sorted: The process is repeated until the list is sorted. Each pass through the list places the next largest element in its correct position.

    Example

    Consider sorting the array [5, 2, 4, 6, 1, 3] using bubble sort:

    • Pass 1: Compare and swap adjacent elements if necessary. After the first pass, the largest element 6 moves to the end.
    • [2, 4, 5, 1, 3, 6]
    • Pass 2: Continue comparing and swapping. After the second pass, the next largest element 5 moves to its correct position.
    • [2, 4, 1, 3, 5, 6]
    • Pass 3: Repeat until sorted. After further passes, smaller elements move towards the beginning, and eventually, the array becomes sorted.
